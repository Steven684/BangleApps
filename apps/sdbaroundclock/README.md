# SdB Around Clock

Just a clock... or is it?  
This is a **clock, that goes around**...!

![](sdbaroundclock.png)

## Usage

It's a clock...
* Upload it like any other clock
* Enable it via Settings > System > Select Clock
* It shows time and day-of-week
* Swipe left/right for date and day-of-week
